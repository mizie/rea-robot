Toy Robot Simulator
===================

How to run 
===========
java -jar RobotREA.jar


valid commands
================
PLACE X,Y,F --> int:x (0 -5) and int:y (0 -5), F : {NORTH|EAST|SOUTH|WEST}
MOVE
LEFT
RIGHT
REPORT
exit

Source code structure
======================
src
   com.rea.toyrobot.simulator
      Board.java
	  Robot.java
	  Simulator.java (main)

Use Java 1.8
